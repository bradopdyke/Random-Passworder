# Password Generator Starter Code

A password generator that lets users make secure passwords based on their preferred criteria. In this project, the JavaScript was written to function properly, as intended with the HTML and CSS. 

Features include:
1) A working generate button
2) Requests password specifications
3) Validates input 
4) Password is written on the page

How to Use: 
1) Select "Generate Password" 
2) Choose the parameters for your password
3) Input is validated.
4) A uninque password will be generated.
